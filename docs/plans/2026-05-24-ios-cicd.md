# iOS CI/CD Implementation Plan

> **For agentic workers:** Implement task-by-task; check off steps as you go. Do not skip prerequisite verification.

**Goal:** GitHub Actions + Fastlane + match pipelines: PR → TestFlight internal (unless `no-deploy`); tag `v*` → App Store submit.

**Architecture:** Thin workflows call Fastlane lanes (`ci_build`, `preview`, `release`). match provides signing from a private cert repo. App Store Connect API key uploads artifacts.

**Tech Stack:** GitHub Actions (`macos-15`), Fastlane, match, Xcode 16+, scheme `MusicWall`, bundle `chris.MusicWall`.

**Spec:** `docs/specs/2026-05-24-ios-cicd-design.md`

### Task 3: Fastlane lanes

**Files:**
- Create: `fastlane/Fastfile`

- [ ] **Step 1:** Add shared helper for API key:

```ruby
def asc_api_key
  app_store_connect_api_key(
    key_id: ENV["APP_STORE_CONNECT_API_KEY_KEY_ID"],
    issuer_id: ENV["APP_STORE_CONNECT_API_KEY_ISSUER_ID"],
    key_content: ENV["APP_STORE_CONNECT_API_KEY_CONTENT"],
    is_key_content_base64: true
  )
end
```

- [ ] **Step 2:** Lane `ci_build` — no match:

```ruby
lane :ci_build do
  scan(
    scheme: "MusicWall",
    destination: "platform=iOS Simulator,name=iPhone 16",
    build_for_testing: false
  ) # OR xcodebuild build if scan unavailable without tests — use:
  # build_app(skip_package_ipa: true, skip_archive: true, destination: ...)
end
```

Use `build_app` with `skip_archive: true` and simulator destination if `scan` requires a test target. Prefer explicit `xcodebuild -scheme MusicWall -destination 'platform=iOS Simulator,name=iPhone 16' build` in a `sh` action if simpler.

- [ ] **Step 3:** Lane `preview`:

```ruby
lane :preview do
  setup_ci if ENV['CI']
  match(type: "appstore", readonly: true)
  increment_build_number(build_number: ENV["BUILD_NUMBER"])
  build_app(
    scheme: "MusicWall",
    export_method: "app-store"
  )
  upload_to_testflight(
    api_key: asc_api_key,
    skip_waiting_for_build_processing: true,
    distribute_external: false
  )
end
```

- [ ] **Step 4:** Lane `release`:

```ruby
lane :release do
  setup_ci if ENV['CI']
  match(type: "appstore", readonly: true)
  version = ENV["RELEASE_VERSION"] # pass from tag without v
  increment_version_number(version_number: version)
  increment_build_number(build_number: ENV["BUILD_NUMBER"])
  build_app(scheme: "MusicWall", export_method: "app-store")
  upload_to_app_store(
    api_key: asc_api_key,
    submit_for_review: true,
    automatic_release: false,
    force: true,
    skip_metadata: true,
    skip_screenshots: true
  )
end
```

Tune `automatic_release` to human preference from spec prereq #6.

### Task 4: PR preview workflow

**Files:**
- Create: `.github/workflows/ios-preview.yml`

- [ ] **Step 1:** Workflow skeleton:

```yaml
name: iOS Preview
on:
  pull_request:
    types: [opened, synchronize, reopened]
jobs:
  preview:
    if: github.event.pull_request.head.repo.full_name == github.repository
    runs-on: macos-15
    steps:
      - uses: actions/checkout@v4
      - uses: ruby/setup-ruby@v1
        with:
          bundler-cache: true
      # ssh agent for match deploy key
      # bundle exec fastlane ci_build OR preview based on label
```

- [ ] **Step 2:** Add step to detect `no-deploy` label:

```yaml
- uses: actions/github-script@v7
  id: labels
  with:
    script: |
      const labels = context.payload.pull_request.labels.map(l => l.name);
      return labels.includes('no-deploy');
```

Or `gh pr view` with `GITHUB_TOKEN`.

- [ ] **Step 3:** Configure SSH for match:

```yaml
- uses: webfactory/ssh-agent@v0.9.0
  with:
    ssh-private-key: ${{ secrets.MATCH_GIT_PRIVATE_KEY }}
```

Add cert repo host to `known_hosts`.

- [ ] **Step 4:** Env vars: `BUILD_NUMBER: ${{ github.run_number }}`, `MATCH_PASSWORD`, `MATCH_GIT_URL`, API key secrets, `CI: true`.

- [ ] **Step 5:** Conditional:

```yaml
- run: bundle exec fastlane ci_build
  if: steps.labels.outputs.result == 'true'
- run: bundle exec fastlane preview
  if: steps.labels.outputs.result != 'true'
```

- [ ] **Step 6:** PR comment step after preview (build number, link to Actions run).
